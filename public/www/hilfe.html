Exponatverwaltung


Kann die Exponatnummer frei vergeben werden?

Ja, die Exponatnummer darf beliebig vergeben werden, muss allerdings mindestens drei Ziffern enthalten und einmalig sein. Vorangestellte Nullen werden automatisch herausgefiltert (001 wird zu 1).

Wie können die Audio- und Bild- Dateien hochgeladen werden?

Audio- und Bild- Dateien können über eine beliebige Cloud freigegeben werden (z.B. Dropbox, Google Drive, MyCloud etc.). Die URL der freigegebenen Datei kann via Inputfeld dem entsprechenden Exponat zugeordnet werden.

Lässt sich die Exponatsübersicht sortieren?

Ja, diese kann nach Exponatname, Nummer, Standort oder Themennummer sortiert werden. Wählen Sie dazu die Tabellenüberschrift an. Sofern diese sich grau verfärbt, lässt sich die Liste gemäss Auswahl sortieren.

Wozu dient das Kennzeichen Aktiv?

Über das Kennzeichen Aktiv wird gesteuert, welche Exponate im Audioguide angezeigt werden (Kennzeichen gesetzt).

Wie kann die Sendereichweite der einzelnen Beacons angepassen werden?

Die Sendereichweite kann über das Feld Sendestärke bestimmt werden. Die Werte bedeuten folgende Reichweiten:

  • -30 rund 50cm bis 1 m
  • -12 rund 10 m
  • 4 bis zu 70 m

Können die Beacons direkt aus dem Verwaltungtool angepasst werden?

Zurzeit werden Ihre Beacons lediglich in Echtzeit angezeigt - somit lässt sich deren Status erfragen. Aktuell bietet der Beacon hersteller noch keine Möglichkeit, die Verwaltung extern durchzuführen - sollte allerdings bald folgen. Um die Beacon-Einstellungen anzupassen, wählen Sie ein Beacon an und Sie werden auf die Webseite des Herstellers weitergeleitet.

Audioguide


Wie kann eine Audioguide-Station erfasst werden?

Über die Major-ID des Beacons lässt dieser sich eindeutig identifizieren. Wenn Sie über das Customizing-Menu einen neuen Standort hinterlegen wollen, clicken Sie auf die entsprechende Stelle auf der Karte. Es wird automatisch ein Marker gesetzt. Schreiben Sie anschliessend den gewünschten Text, welcher auf der Audioguide-Karte angezeigt wird in das Feld "Name" und die Major- ID des entsprechenden Beacons in das Feld "Beacon-Nr." Nachdem die Schaltfläche Speichern gedrückt wurde, ist der Ort gespeichert und auf dem Audioguide angepasst.


Gibt es weitere Customizing- Einstellungen?

Aktuell lässt sich nur das Logo sowie die Karte des Museums anpassen. Wenn Sie weitere Anpassungswünsche haben, kontaktieren Sie mich bitte.
